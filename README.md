Display made with X11 libraries in C.

## Linux:
Run `make` to build.

`./simwall -h` for command line usage. 


Keybinds:

`Ctrl-Alt-Q` Quit

`Ctrl-Alt-P` Pause

`Ctrl-Alt-A` Enter/Exit add mode

Add Mode:

    - Hold the left mouse button to add cells
